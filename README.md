# ExCombo

Node-based job rotation flow editor for FFXIV (Dalamud plugin).

Design your job rotation as a visual node graph. Flows replace hotbar actions
in-game as you build them. Use `/excombo` to open the editor.

## Install

1. In-game, open Dalamud settings: `/xlsettings`
2. **Experimental** tab → **Custom Plugin Repositories**
3. Add this URL and enable it:

   ```
   https://raw.githubusercontent.com/nixglacie/ExCombo-dist/main/pluginmaster.json
   ```

4. Save. Open the plugin installer (`/xlplugins`), search **ExCombo**, install.

This repository distributes the compiled plugin only.
